# READING LIST APPLICATION

This app allows users to track the books they waht to read

#### Instructions
.Download the source code
```bash
git clone https://github.com/00012876/ReadingList.git
```

or just download the zip file

2.Install deps
```bash
npm install
```
3.Run the app
```bash
node app
```
